To Install:

Elefront.gha
Drag and drop onto our grasshopper canvas to install.
If you are using both v5 andd v6 on your machine. Copy Elefront.gha to the following location (XXX is Windows user name): 
C:\Users\XXX\AppData\Roaming\Grasshopper\6\Libraries

ElefrontProperties.rhp
Drag and drop onto your Rhino model view to install.
This is a object property tab in Rhino. Click on any object and the tab will be available at the Object Properties panel.
